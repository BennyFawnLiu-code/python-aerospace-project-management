# python-aerospace-project-management
基于Python开发的中国航天项目智能管理系统，整合嫦娥探月、长征火箭、高分卫星、北斗导航等35+真实航天项目数据。核心功能包含项目信息增删改查（CSV文件存储，实现数据持久化）、数据分析（所属机构分布、项目持续时间统计、成果状态分类等）、8类可视化展示（饼图、雷达图、词云、发射基地地理分布图等），并内置航天主题射击小游戏。采用pandas、matplotlib、pyecharts、pygame等库开发，是Python初学者的综合实践作品，集中体现文件操作、第三方库集成、终端UI设计等核心技能。部署简单、操作直观，适合Python学习交流或展示基础开发能力。
