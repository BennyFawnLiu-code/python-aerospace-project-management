## 项目简介
本项目是大一刚完成Python基础学习后，开发的**全功能航天项目智能管理系统**。系统以中国航天事业发展为核心，整合了35个真实航天项目的完整数据（涵盖探月工程、运载火箭、遥感卫星、导航系统、空间站、深空探测等6大类别），提供从数据管理、统计分析、可视化展示到趣味互动的一站式服务。

项目无需复杂数据库部署，采用CSV文件实现数据持久化存储，支持离线使用；终端UI界面通过颜色分区和清晰菜单设计，操作直观易懂；整合多个Python第三方库，实现了理论知识到实际应用的落地，是Python入门阶段的综合性实践作品，适合用于学习交流、课程展示或技能积累。

### 项目定位
- 面向人群：Python初学者、航天爱好者、需要数据管理/可视化练习的学生
- 核心价值：整合真实数据，覆盖Python文件操作、库调用、UI设计、数据分析、可视化等核心技能
- 适用场景：课程设计、技能练习、航天数据整理、Python入门作品展示

## 核心数据详情
系统内置35个真实中国航天项目数据，涵盖1965年至今的重大航天工程，数据字段完整、来源可靠，具体分类及核心项目示例如下：

### 项目分类（6大类别）
1. **探月工程**（7个）：嫦娥一号至六号、嫦娥五号T1（绕月探测、月面着陆、采样返回等）
2. **运载火箭**（7个）：长征一号至六号、长征十一号（液体/固体燃料、不同运力等级）
3. **遥感卫星**（14个）：高分一号至十四号（光学/雷达/高光谱、高分辨率对地观测）
4. **导航与通信**（2个）：北斗卫星导航系统、鸿雁星座（全球导航、卫星通信）
5. **空间站与载人航天**（5个）：天和核心舱、问天实验舱、梦天实验舱、中国空间站、神舟系列
6. **深空与专项探测**（3个）：天问一号（火星）、羲和号（太阳）、悟空号（暗物质）

### 数据时效性
- 已完成项目：包含完整启动/完成年度（如嫦娥五号2020年启动并完成）
- 持续运行项目：明确标注状态（如中国空间站、高分系列卫星）
- 数据覆盖周期：1965年（长征一号启动）-2024年（嫦娥六号完成）

## 核心功能模块
系统包含10大核心功能，覆盖数据管理、分析、可视化、互动四大场景，每个功能均经过实际测试，稳定可用：

### 1. 项目信息管理（基础核心）
- 🔍 多条件查询：支持按「启动年度」「项目类型」「所属机构」精准查询
- ✏️ 全字段修改：可修改项目名称、启动/完成年度、成果状态、所属机构、项目类型
- ➕ 新增项目：自动校验项目编号唯一性，支持自定义项目信息录入
- 🗑️ 安全删除：按项目编号删除，删除前无需额外确认（简化操作，适合初学者）
- 📋 排序展示：支持按启动年度升序/降序排列，表格自动对齐，字段颜色区分

### 2. 数据分析（统计核心）
- 🏢 所属机构统计：统计各机构承担的项目数量（如中国航天科技集团、中国科学院等）
- 📊 成果状态统计：区分「已完成」和「持续运行」项目的数量分布
- ⏳ 项目持续时间分析：计算有效项目（启动/完成年度完整）的平均时长、最大值、最小值、中位数、方差等核心指标
- 📈 详细统计报告：输出结构化统计结果，支持直接复制使用

### 3. 数据可视化（展示核心）
8种可视化图表，覆盖饼图、折线图、散点图、词云、地理分布等多种类型，图表支持保存和放大查看：
- 机构分布饼图：直观展示各机构项目占比
- 发射数量趋势图：1970-2024年中国航天器发射数量变化
- 项目持续时间散点图：呈现启动年度与持续时间的关联
- 航天主题词云：基于项目名称/类型生成中国地图形状词云
- 发射基地地理分布图：展示西昌、酒泉、文昌等发射基地分布
- 图片轮播：展示23张航天主题图片（空间站、火箭、卫星等）
- 机构项目柱状图：对比各机构项目数量差异
- 项目类型雷达图：多维度展示不同类型项目分布

### 4. 趣味互动（拓展功能）
- 🎮 航天主题射击游戏：方向键控制飞船移动，空格键发射子弹，躲避陨石和敌机
- 🔊 音效反馈：射击、爆炸均有对应音效，增强沉浸感
- 📝 得分统计：实时显示游戏得分，击中陨石得10分，击中敌机得20分

### 5. 数据持久化
- 自动保存：新增、修改、删除项目后，数据自动写入CSV文件，关闭程序不丢失
- 手动保存：支持主动触发保存操作，双重保障数据安全
- 数据格式兼容：CSV文件可直接用Excel打开编辑（需注意编码格式）

## 详细环境搭建
### 第一步：安装Python（必选）
#### Windows系统
1. 下载Python 3.8.10安装包：[官网下载链接](https://www.python.org/downloads/release/python-3810/)
2. 双击安装包，勾选「Add Python 3.8 to PATH」（关键！自动添加环境变量）
3. 选择「Customize installation」，保持默认选项，点击「Next」
4. 勾选「Install for all users」，选择安装路径（建议默认C:\Python38），点击「Install」
5. 验证安装：打开CMD命令提示符，输入 `python --version`，显示「Python 3.8.10」即为成功

#### MacOS系统
1. 下载Python 3.8.10安装包：[官网下载链接](https://www.python.org/downloads/release/python-3810/)（选择macOS 64-bit installer）
2. 双击安装包，按照引导完成安装（默认会自动添加环境变量）
3. 验证安装：打开终端，输入 `python3 --version`，显示「Python 3.8.10」即为成功

#### Linux系统（Ubuntu）
1. 打开终端，输入以下命令安装Python 3.8：
   ```bash
   sudo apt update
   sudo apt install python3.8 python3.8-pip
   ```
2. 验证安装：输入 `python3.8 --version`，显示「Python 3.8.x」即为成功

### 第二步：安装依赖库（必选）
1. 打开终端/CMD，输入以下命令（国内用户推荐使用清华镜像源加速，避免超时）：
   ```bash
   # Windows系统（使用python和pip命令）
   pip install pandas==1.5.3 numpy==1.23.5 matplotlib==3.7.1 wordcloud==1.8.2.2 pyecharts==2.0.3 jieba==0.42.1 pygame==2.1.3 colorama==0.4.6 pillow==9.4.0 -i https://pypi.tuna.tsinghua.edu.cn/simple

   # Mac/Linux系统（使用python3和pip3命令）
   pip3 install pandas==1.5.3 numpy==1.23.5 matplotlib==3.7.1 wordcloud==1.8.2.2 pyecharts==2.0.3 jieba==0.42.1 pygame==2.1.3 colorama==0.4.6 pillow==9.4.0 -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```
2. 验证安装：输入 `pip list`（Windows）或 `pip3 list`（Mac/Linux），查看上述库是否均已安装成功（无红色报错即可）

### 第三步：安装中文字体（解决可视化乱码，可选但推荐）
#### Windows系统
默认已安装「黑体（SimHei）」，无需额外操作（代码已配置优先调用该字体）

#### MacOS系统
1. 下载黑体字体文件（SimHei.ttf）：[网盘链接](https://pan.baidu.com/s/1qZxX3yWZ8cY6X7aZ9bY8cZ)（提取码：h52t）
2. 打开字体册（Font Book），点击「文件」→「添加字体」，选择下载的SimHei.ttf安装
3. 重启终端，确保字体生效

#### Linux系统（Ubuntu）
1. 下载黑体字体文件：
   ```bash
   wget https://mirrors.tuna.tsinghua.edu.cn/help/fonts/SimHei.ttf
   ```
2. 复制字体到系统字体目录：
   ```bash
   sudo cp SimHei.ttf /usr/share/fonts/
   ```
3. 更新字体缓存：
   ```bash
   sudo fc-cache -fv
   ```
4. 验证字体：输入 `fc-list | grep SimHei`，显示字体路径即为成功

### 第一步：下载项目
1. 打开GitHub仓库地址：[你的仓库链接，如https://github.com/你的用户名/python-aerospace-project-management]

1. 打开CMD命令提示符，切换到项目解压目录（示例路径，需替换为你的实际路径）：


### 第三步：功能使用详解（按菜单序号）
#### 1. 显示所有项目
- 操作：输入「1」→ 按回车 → 选择排序方式（1.升序/2.降序/3.不排序）→ 按回车
- 效果：显示结构化表格，包含35个项目的所有字段，不同字段用不同颜色区分（序号白色、项目名称绿色、年度黄色等）
- 退出：查看完成后，按任意键返回主菜单

#### 2. 添加新项目
- 操作：输入「2」→ 按回车 → 依次输入项目编号、名称、启动年度、完成年度（可空）、成果状态、所属机构、项目类型 → 按回车
- 校验规则：项目编号不可重复，必填字段（编号、名称、启动年度等）不可为空
- 保存：添加完成后自动保存到CSV文件，显示「添加成功」→ 按任意键返回

#### 3. 删除项目
- 操作：输入「3」→ 按回车 → 输入要删除的项目编号（如「嫦娥一号」）→ 按回车
- 校验规则：若输入的编号不存在，显示「项目不存在」；存在则直接删除
- 保存：删除后自动保存到CSV文件，显示「删除成功」→ 按任意键返回

#### 4. 修改项目信息
- 操作：输入「4」→ 按回车 → 输入要修改的项目编号 → 按回车 → 依次输入新字段值（留空表示不修改）→ 按回车
- 特点：支持单个字段修改，无需重复输入其他字段
- 保存：修改后自动保存到CSV文件，显示「修改成功」→ 按任意键返回

#### 5. 保存文件
- 操作：输入「5」→ 按回车
- 作用：手动触发保存（增删改操作已自动保存，此功能用于手动备份）
- 反馈：显示「数据已保存！」→ 按任意键返回

#### 6. 查询项目
- 操作：输入「6」→ 按回车 → 选择查询条件（1.启动年度/2.项目类型/3.所属机构）→ 按回车 → 输入查询关键词 → 按回车
- 示例：查询「中国国家航天局」的项目 → 选择「3.所属机构」→ 输入「中国国家航天局（CNSA）」→ 显示所有相关项目
- 反馈：若查询结果为空，显示「未找到匹配项目」；否则显示结构化结果 → 按任意键返回查询菜单（可继续查询或输入「4」返回主菜单）

#### 7. 数据分析
- 操作：输入「7」→ 按回车 → 选择分析类型（1.所属机构统计/2.成果状态统计/3.项目持续时间分析/4.详细统计指标/5.返回主菜单）→ 按回车
- 示例：选择「3.项目持续时间分析」→ 显示参与计算的项目列表及平均持续时间；选择「4.详细统计指标」→ 显示平均值、最大值、最小值等
- 反馈：若数据不完整（如无有效启动/完成年度），显示「没有有效数据可计算」→ 按任意键返回

#### 8. 数据可视化
- 操作：输入「8」→ 按回车 → 选择可视化类型（1-8）→ 按回车
- 各功能详情：
  1. 机构分布饼图：自动生成各机构项目占比饼图，支持放大/保存
  2. 发射数量趋势：显示1970-2024年航天器发射数量折线图，标注具体数值
  3. 持续时间散点图：以启动年度为X轴、持续时间为Y轴，呈现数据分布

## 游戏玩法说明
### 游戏目标
控制飞船躲避陨石和敌机，通过发射子弹摧毁目标，获取尽可能高的分数

### 操作指南
| 操作按键   | 功能                     |
|------------|--------------------------|
| 方向键「←」| 飞船向左移动（每次移动8像素） |

- 摧毁陨石：+10分/个

### 游戏设置
- 窗口大小：1000×1200像素
- 帧率（FPS）：60帧/秒
- 敌人数量：初始3个陨石、5个敌机，摧毁后自动刷新
- 音效：射击、爆炸均有对应音效，支持系统音量调节

## 常见问题与解决方案
### 问题1：运行 `python main.py` 提示「ModuleNotFoundError: No module named 'xxx'」
- 原因：对应的依赖库未安装成功，或安装路径未添加到环境变量
- 解决方案：
  1. 确认输入的命令正确（Windows用pip，Mac/Linux用pip3）
  2. 重新执行依赖库安装命令，确保无红色报错
  3. 若仍失败，手动安装单个库（如 `pip install xxx==版本号`）
  4. 检查Python环境变量是否配置正确（可重新安装Python并勾选「Add to PATH」）

### 问题2：可视化图表中文显示为方块/问号
- 原因：系统缺少黑体（SimHei）字体，或代码中字体路径配置错误
- 解决方案：
  1. 按照「详细环境搭建」中的「安装中文字体」步骤操作
  2. 若仍无效，修改 `main.py` 中可视化相关代码的字体配置：
     ```python
     # 将 plt.rcParams['font.sans-serif'] = ['SimHei'] 改为系统已有的中文字体名称
     # 例如Mac系统可改为：
     plt.rcParams['font.sans-serif'] = ['Arial Unicode MS']
     # Linux系统可改为：
     plt.rcParams['font.sans-serif'] = ['WenQuanYi Zen Hei']
     ```
  3. 重启系统，重新生成图表

### 问题3：游戏启动失败，提示「FileNotFoundError: No such file or directory: 'img/background.png'」
- 原因：`img/` 或 `sound/` 文件夹缺失，或文件路径错误
- 解决方案：
  1. 检查项目根目录是否存在 `img/` 和 `sound/` 文件夹
  2. 确认文件夹内包含所有必要文件（如background.png、shoot.wav等）
  3. 若文件缺失，重新下载项目压缩包，确保解压完整
  4. 确保项目路径无中文（如避免「D:\Python项目\」，改为「D:\PythonProjects\」）

### 问题4：读取CSV文件时提示「UnicodeDecodeError: 'utf-8' codec can't decode byte 0xb7 in position 10: invalid start byte」
- 原因：CSV文件编码不兼容（项目默认使用GBK编码，而系统默认用UTF-8读取）
- 解决方案：
  1. 用记事本打开报错的CSV文件（如「航天项目信息表1.csv」）
  2. 点击「文件」→「另存为」，在「编码」下拉框中选择「ANSI」（Windows）或「GBK」（Mac）
  3. 覆盖原文件，重新运行系统
  4. 若仍报错，检查 `main.py` 中读取CSV的代码是否指定编码：
     ```python
     # 确保代码中包含 encoding="gbk"
     with open(fileName, "r", encoding="gbk") as fo:
     ```

### 问题5：终端UI界面显示错乱（文字重叠、边框不完整）
- 原因：终端窗口大小不足，或字体设置导致排版异常
- 解决方案：
  1. 放大终端窗口（拖动窗口边缘扩大宽度和高度）
  2. 调整终端字体大小（Windows CMD：右键窗口标题→「属性」→「字体」→ 选择「小字体」；Mac终端：「终端」→「设置」→「字体」→ 调整字号为12号）
  3. 重启系统，重新加载UI界面

### 问题6：词云生成失败，提示「FileNotFoundError: [Errno 2] No such file or directory: '中国2.png'」
- 原因：词云模板图片「中国2.png」缺失
- 解决方案：
  1. 检查项目根目录是否存在「中国2.png」文件
  2. 若缺失，重新下载项目压缩包
  3. 确保文件名正确（无拼写错误，如「中国2.png」而非「中国.png」）
